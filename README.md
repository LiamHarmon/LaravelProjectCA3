## Create Database 
```
mysql
create database laravelblog
exit;
```
## Setup your database credentials in the .env file
```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravelblog
DB_USERNAME={USERNAME}
DB_PASSWORD={PASSWORD}
```
## Migrate the tables
```
php artisan migrate
```
